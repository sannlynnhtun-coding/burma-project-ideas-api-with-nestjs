Structured cultural data for products, research, and learning tools. ထုတ်ကုန်များ၊ သုတေသနနှင့် သင်ယူရေးကိရိယာများအတွက် စနစ်တကျဖွဲ့စည်းထားသော ယဉ်ကျေးမှုဒေတာ။

Burma Project Ideas API brings curated Myanmar cultural, educational, travel, food, calendar, and reference datasets into one documented REST surface. Burma Project Ideas API သည် စိစစ်စုစည်းထားသော မြန်မာ့ယဉ်ကျေးမှု၊ ပညာရေး၊ ခရီးသွား၊ အစားအစာ၊ ပြက္ခဒိန်နှင့် ကိုးကားဒေတာများကို စာရွက်စာတမ်းပြည့်စုံသော REST API တစ်ခုအဖြစ် ပေးထားသည်။

Documentationစာရွက်စာတမ်း

Choose the API explorer that fits your workflow. သင့်လုပ်ငန်းစဉ်နှင့် ကိုက်ညီသော API explorer ကိုရွေးချယ်ပါ။

Both documentation surfaces stay at their existing routes and expose the same NestJS OpenAPI specification. စာရွက်စာတမ်းနှစ်မျိုးလုံးသည် လက်ရှိ route များတွင်ရှိပြီး တူညီသော NestJS OpenAPI specification ကိုဖော်ပြပေးသည်။

  • Swagger API Docs

    Inspect endpoints, schemas, parameters, and responses in the classic OpenAPI interface.OpenAPI interface တွင် endpoint၊ schema၊ parameter နှင့် response များကိုစစ်ဆေးပါ။

API CatalogAPI စာရင်း

Curated data domains in one predictable API surface. ခန့်မှန်းရလွယ်သော API တစ်ခုထဲမှ စိစစ်စုစည်းထားသည့် ဒေတာကဏ္ဍများ။

Each group is organized as a focused resource so apps can discover content without custom scraping or one-off data files. App များက custom scraping သို့မဟုတ် တစ်ခါသုံး data file မလိုဘဲ အကြောင်းအရာရှာနိုင်ရန် အုပ်စုတစ်ခုစီကို သီးခြား resource အဖြစ်ဖွဲ့ထားသည်။

Capabilitiesစွမ်းဆောင်ရည်များ

Designed for teams that need clear data contracts. ရှင်းလင်းသော ဒေတာစာချုပ်များလိုအပ်သည့် အဖွဲ့များအတွက် ဒီဇိုင်းပြုထားသည်။

The landing page presents the API as an operational data product while leaving the NestJS implementation intact. Landing page သည် NestJS implementation ကိုမပြောင်းဘဲ API ကိုအသုံးချနိုင်သော ဒေတာထုတ်ကုန်တစ်ခုအဖြစ် ဖော်ပြထားသည်။

  • Stable routesတည်ငြိမ်သော route များ

    Docs and API URLs remain predictable across the existing deployment surface.Docs နှင့် API URL များသည် လက်ရှိ deployment တွင် ခန့်မှန်းရလွယ်ကူစွာ တည်ငြိမ်နေသည်။

  • Curated datasetsစိစစ်စုစည်းထားသော ဒေတာစုများ

    Static public data sources are presented as organized REST resources.Static အများသုံးဒေတာရင်းမြစ်များကို စနစ်တကျဖွဲ့ထားသော REST resource များအဖြစ် ပေးထားသည်။

  • Fast entry pointလျင်မြန်သော ဝင်ပေါက်

    The home page stays lightweight with no frontend framework or CDN dependency.Home page တွင် frontend framework သို့မဟုတ် CDN dependency မရှိသဖြင့် ပေါ့ပါးနေသည်။

  • Accessible UIလူတိုင်းအသုံးပြုနိုင်သော UI

    Keyboard focus, contrast, responsive layouts, and motion preferences are built in.Keyboard focus၊ အရောင်ကွာခြားမှု၊ responsive layout နှင့် motion preference များပါဝင်သည်။
